IOT Smart Gas Detection System

Introduction

It is an (IOT) related Gas Detection System, which help to detect the leakage of the gas and inform the relevant person. Our aim is to reduce the risks in Kitchen using Internet of Things. The main aim is to propose the design and construction of an SMS based Gas Leakage Alert System. Gas sensor are used to detect gas leakages in a kitchen. With the help of this sensor the issue of gas wastage is also monitored. The sensor keeps on recording the data continuously and if it is found any leakage of gas, the sensor sends the data to the microcontroller, microcontroller analysis the data recorded by the sensor and check whether the leakage of the gas is in safe state or danger state. if system found the gas leakage is continuously increasing than it pushes the alarm buzzer which can make some sound to inform the home guardian. It also able to send the SMS to the home owner who is connected with the system to get an awareness message.

Major Components

Raspberry Pi-3

The Raspberry Pi 3 is a tiny credit card size computer. Just add a keyboard, mouse, display, power supply, micro SD card with installed Linux Distribution and you'll have a fully fledged computer that can run applications from word processors and which is going to used processing the sensor inputs.

SIM Module

The SIM900 is a complete Quad-band GSM/GPRS solution in a SMT module which can be embedded in the customer applications. Featuring an industry-standard interface, the SIM900 delivers GSM/GPRS 850/900/1800/1900MHz performance for voice, SMS, Data, and Fax in a small form factor and with low power consumption. With a tiny configuration of 24mm x 24mm x 3 mm, SIM900 can fit almost all the space requirements in your M2M application, especially for slim and compact demand of design.

Sensor

This is a simple-to-use liquefied petroleum gas (LPG) sensor, suitable for sensing LPG (composed of mostly propane and butane) concentrations in the air. The MQ-6 can detect gas concentrations anywhere from 200 to 10000ppm. ... The sensor's output is an analog resistance.
